# vdir-cli

Vdir is a way to organize your files in a way
want to see them, regardless of how they need to be on the file.
You can use Folders, References and importantly, Queries.
Queries are the reason this tool exists.
A query has a provider which returns a list of filenames, usually, a filter on files in a scope.
You can use f.e. ripgrep to find all files which have 'TODO:' in them!
